Quick Workflows

  1. Add a patient: click Add -> complete modal fields -> Save.
  2. Filter current census: set Date/Hospital/Room/Patient -> click Apply -> review rows.
  3. Run analytics: open Analytics tab -> choose preset -> Run Preset (or enter valid custom query).
  4. Create PDF report: open Reports tab -> choose type/date/hospital -> Generate PDF.
  5. Prepare handoff: select records in Census -> click Handoff in footer -> Copy from modal.
